The Document Object Model (DOM)

How do you select an element from the DOM?

You can select elements from the DOM using various methods in JavaScript:getElementById(),getElementsByClassName(),querySelector()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element instead of adding individual listeners to each child element.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate attributes and styles of elements in the DOM using various methods:setAttribute(attributeName, value), getAttribute(attributeName)
